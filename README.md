# EasyDev
A RP2040 Dev Board featuring 43 exposed pins along with 16 MB of flash storage! Can support up to (not recommended) 1A of power draw on VBUS! Features 29 programmable GPIO pins so that you have plenty of pins for plenty of connecting and tinkering!

# Why?
I was thinking of making some PCBs, and I thought, why not try something advanced and also something that you have been wanting to do for a while now? And that's when I came up with the idea of making my own RP2040 dev board (basically a Raspberry Pi Pico) starting from scratch. So I did, and learned a lot along the way! Making this devboard will definitely change the way I make PCBs in the future especially because of all the things that I learned from designing this.

# BOM
The only part needed to be ordered for this project is the PCB with PCBA parts on it. If you would like to see, go check out the EasyDev-bom.csv file as that is the same one that I used for getting my PCBA and has all the parts needed. If you would like the JLCPCB version of the BOM with parts picked out that will already be set up with JLCPCB's BOM, go see EasyDev-bom-jlc.xls!
